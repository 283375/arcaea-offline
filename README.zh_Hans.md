## 这是什么？

这是 `Arcaea Offline` 的核心依赖库，用于维护分数数据库、计算潜力值，并提供一些实用工具。

## 这怎么用？

该仓库是一个 python 库。

对普通用户，如果你不知道“库”是什么，你应该对[这个 GUI](https://github.com/283375/arcaea-offline-pyside-ui) 更感兴趣。

对开发者，文档仍在建设当中，敬请期待。
